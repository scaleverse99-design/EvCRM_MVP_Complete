The Sovereign Cloud Suite

SmartNotes

An automated browser extension that captures and parses conversations with ChatGPT, Claude, and Gemini, writing notes directly into your private Google Drive storage vault.

  • Private Baremetal: Stored on your own Google Drive folder index.
  • Auto-Indexing: Directory database lists, saves, and parses chat outputs.
  • Code Snippets Extraction: Converts session logs into clean markdown.

Capsule

A secure context handoff utility. Extract discussion files, tech keywords, and next actions into a formatted context capsule, bridging web AI chats and your local IDE.

  • 100% Client-Side: Manual copy-paste handoff in both directions.
  • Smart Heuristics: Programmatic summary, task, and path parser.
  • IDE SQLite Sync: SQLite reader copies logs lock-free from state tables.
Download Chrome Extension (.zip)

Chrome Setup Steps

1 Download and extract the capsule-extension.zip file.
2 Open chrome://extensions in your Chrome browser.
3 Toggle the Developer mode switch in the top-right corner.
4 Click Load unpacked (top-left) and select the extracted folder.
Download VS Code / Cursor (.vsix)

IDE Setup Steps

1 Download the capsule-vscode-1.0.0.vsix package.
2 In Cursor/VS Code, open the Extensions view (Ctrl+Shift+X).
3 Click the ... (More Actions) button on the side panel.
4 Choose Install from VSIX... and select the downloaded file.

The Sovereign Design System

Zero Central Database

We never host or sync your files to centralized databases. All operations process locally inside your client environment.

Google Drive Baremetal

All directories, indices, and session summaries are written directly into your private Google Drive vault via client-authenticated API tokens.

Manual Security

No auto-injections into destination pages. Clipboard data is managed, reviewed, and pasted manually by you to prevent malicious context poisoning.
